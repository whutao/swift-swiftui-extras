# Changelog

## [1.1.0] - 2025-11-03

### Added

- EnvironmentValues extensions.
- `Star` shape.
- `AnimatableVector` type.
- `DividedVStack` and `DividedHStack` views.
- `ShapeBuilder` and `InsettableShapeBuilder` types.

## [1.0.0] - 2024-10-22

### Added

- Environment, Binding, EdgeInsets, View extensions.
- `AnimatableTriple`, `AnimatableQuadruple` types.
- `WithInitialValue` view utility.
